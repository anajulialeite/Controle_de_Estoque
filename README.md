O **EstoqueMaster** é um sistema completo e intuitivo para controle e gestão de estoque de mercadorias. Ele foi projetado para auxiliar no gerenciamento de inventário de pequenas e médias empresas, permitindo cadastrar categorias, fornecedores e produtos, além de acompanhar todas as movimentações de entrada e saída com relatórios financeiros automáticos e alertas inteligentes de reposição.

---

## 🛠️ Tecnologias Utilizadas

- **Backend**:
  - **Java 21** (JDK 21)
  - **Spring Boot 4.1.0**
  - **Spring MVC** (Desenvolvimento web baseado em controladores e views)
  - **Spring Data JPA** (Persistência de dados com Hibernate)
  - **H2 Database** (Banco de dados relacional em memória para desenvolvimento ágil)
- **Frontend**:
  - **Thymeleaf** (Engine de template para renderização dinâmica no lado do servidor)
  - **Bootstrap 5** & **Bootstrap Icons** (Design responsivo e componentes modernos)
  - **Chart.js** (Renderização de gráficos interativos no lado do cliente)
  - **Outfit (Google Fonts)** (Tipografia moderna e limpa)
- **Testes**:
  - **JUnit 5** & **Mockito**
  - **Spring Boot Test**
